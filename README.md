# packages-nix

Nix packages, development shells, and NixOS dotfiles configuration.

## Packages

- **blackbox** - Custom Nix package
- **jflap** - JFLAP 7.1 (Java Formal Languages and Automata Package)
- **multibugs** - Custom Nix package
- **openbugs** - Custom Nix package
- **pgquarrel** - PostgreSQL schema comparison tool
- **ytui-music** - YouTube music TUI

## Development Shells

Nix flake-based development environments in `shells/`.

## Dotfiles

Modular NixOS configuration with:

- **Desktop**: bspwm, stumpwm, rofi
- **Browsers**: brave, firefox, qutebrowser
- **Terminals**: alacritty, st
- **Editors**: emacs, vim
- **Dev**: cc, clojure, common-lisp, lua, node, python, rust, scala, shell
- **Shell**: fish, zsh, tmux, git, gnupg
- **Services**: docker, nginx, gitea, jellyfin, wireguard, syncthing, vaultwarden
- **Hardware**: audio, bluetooth, nvidia, wifi, laptop
- **VMs**: lxd, qemu, virtualbox
- **Themes**: alucard (dracula-based)
